## 기능
- 문자열 밀어내기? 이건 의미가 있는지 잘모르겠음
- 문자열로 수식을 넣어주면 알아서 처리해주는 기능

## 요구 사항
- 객체 지향적인 프로그래밍을 필요로 할 것
- 상속 관계를 이용해 기능을 구현할 것
- 사용자의 입력에 대한 오류를 방지하기 위한 코드를 짤 것
- 추가로 프로그래머가 할 수 있는 실수를 방지하는 코드를 짤 것
- 주석을 최대한 활용할 것
